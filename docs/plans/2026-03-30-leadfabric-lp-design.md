## ページ構成（10セクション）

### ① Nav（固定）
- 左：LeadFabricロゴ（ドットグリッドアイコン）
- 中：ナビリンク（課題 / ソリューション / PoC募集）
- 右：`PoC申込はこちら →`（プライマリボタン）

### ② Hero
- バッジ：`● 3社限定 · 無償PoC募集中`（点滅アニメーション）
- H1：「リード情報の汚さが、CRM全体を壊している。」
- サブ：展示会・セミナーで獲得したリードデータを、エンジニア不要でMAやCRMに渡せる状態に自動変換する説明
- ボタン：`無償PoCに申し込む →`（primary）/ `LeadFabricとは？`（outline）

### ③ 課題（Problem）
- 見出し：「こんな状況、ありませんか？」
- 5枚カード：
  1. 展示会後のデータ加工がいつも手作業
  2. エンジニアに頼まないと前に進まない
  3. CRMへのデータ品質が低く、運用が崩壊
  4. インサイドセールスがCRMに入力しない
  5. マーケ投資のROIが経営に説明できない

### ④ 根本原因（Root Cause）
- 見出し：「問題の根本はリード情報の"上流"にある」
- 崩壊フロー図：マーケ（上流）→ インサイド/営業（中流）→ CRM/経営（下流）
- 説明文：上流のデータ品質が後工程全体に影響する構造説明

### ⑤ ソリューション（Solution）
- 見出し：「LeadFabricは、上流を自動で整える」
- フロー図：CSV → LeadFabric（6ステップ自動クレンジング）→ MA/CRM ready
- 3つの価値：エンジニア不要 / 高速変換 / MA・CRM連携対応

### ⑥ 機能（Features）
- 4枚カード：
  1. 会社名正規化・名寄せ（表記揺れ統一・重複検知）
  2. 役職・部門コード自動付与（L1〜L6 / SALES・MKT等）
  3. 業種・企業規模コード自動判定
  4. Marketo互換CSV出力（UTF-8 BOM付き）

### ⑦ PoCオファー
- 見出し：「今だけ3社限定、無償でご提供します」
- 2カラム比較：
  - **提供内容**：LeadFabricフル機能 / 個別サポート / セットアップ支援
  - **お願いすること**：実リードCSVでの検証 / フィードバック / 30〜60分ヒアリング

### ⑧ FAQ
- Q: 現在の価格は？ → A: PoC期間中は完全無償。商用化時に別途ご案内。
- Q: どんなCSV形式に対応？ → A: 展示会・セミナー・Sansan等、任意フォーマットに対応。
- Q: 導入に何が必要？ → A: ブラウザのみ。インストール・エンジニア不要。
- Q: いつから使える？ → A: 申込後、最短1週間以内にご利用開始。

### ⑨ 最終CTA
- 見出し：「まず話を聞いてみませんか。」
- CTA：`Googleフォームで申し込む →`（大ボタン）
- 補足：「残り3社 · 完全無償 · NDA締結の上でご提供」
